Guest File Upload Help

Below are details of how to create your guest list for this site.

1. Download the Guest List Template.
The template contains headings for the details you need to fill in.
N.B. The minimum you must fill in at this stage is the guest first name, last name and postcode.

2.Please ensure if the guest has a 'plus one' that you put "TRUE" in that column.

3. Save the file. You can change the name as long as it ends .csv

4. Upload the file you saved on the Guests Maintenance page.

When you upload this file the system will create a group id which will become the guest's username and their postcode will become their password. You will need to tell the guests their group id/username on their invitation and that their password is the postcode you sent the invitation to.
