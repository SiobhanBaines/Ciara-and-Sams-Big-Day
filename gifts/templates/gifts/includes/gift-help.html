Gift File Upload Help

Below are details of how to create your gift list for this site.

1. Download the Gift List Template.
The template contains headings for the details you need to fill in.

2. Save the file. You can change the name as long as it ends .csv

3. Upload the file you saved on the Gift Maintenance page.

N.B. The images will need to be uploaded individually by going to the Edit page on each item.

Please remove any accents, commas and special characters before uploading
