Menu File Upload Help

Below are details of how to create your Wedding Breakfast Menu.

1. Download the Wedding Breakfast Menu.
The template contains headings for the details you need to fill in.

2. Save the file. You can change the name as long as it ends .csv

3. Upload the file you saved on the Menu Maintenance page.

N.B. Please remove any accents, commas and special characters before uploading.
The course needs to be 'starter', 'main' or 'dessert' for the menu to appear in the correct order.
